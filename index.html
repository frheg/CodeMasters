Game Description

You are trapped in a dungeon and have to find a way out! Kill all the friendly-looking enemies to free yourself. There will be multiple waves of enemies. Can you survive them all?

How to Play

1. Download the game from the GitHub repository.
2. Follow the instructions in the README file to set up and run the game.
3. Use the arrow keys or WASD keys to navigate and mousepad to attack.
4. Have fun and see how long you can survive!

Development Blog

Welcome to the development blog for our game project! Finally we have reached the end of the project and we are excited to share our journey with you. Below, you can find a summary of our progress and key milestones throughout the development process.

Initial Planning
Our team gathered to brainstorm ideas for the game. We decided on a dungeon escape theme with multiple waves of enemies. The planning phase involved outlining the game's mechanics, levels, and initial design.

Prototyping
We started prototyping the basic game mechanics. This included character movement, basic enemy AI, and initial level design. Early prototypes helped us identify key challenges and areas for improvement.

Art and Design
Our game-artist Sara began creating the visual assets for the game. This included character sprites, enemy designs, and environmental elements. The game's visual style started to take shape.

Development and Testing
The development team continued to implement game mechanics and integrate the art assets. We also began internal testing to identify and fix bugs. Playtesting sessions provided valuable feedback for refining the gameplay experience.

Finalizing the Game
As we approached the deadline, we focused on polishing the game. This included balancing the difficulty, adding sound effects and music, and ensuring a smooth user experience.
